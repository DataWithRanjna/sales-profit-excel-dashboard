# Sales & Profit Analysis Dashboard (Excel)

## Overview

This project analyzes sales and profit performance across regions, products, and salesmen using Microsoft Excel.

## Dataset

* Customer-level dataset (1300+ records)
* Includes Region, Product, Sales, and Profit

## Key Insights

* Henry leads in total sales, while Calvin generates the highest profit
* North region drives the highest sales, but Middle region delivers the highest profit
* West region is the weakest performer across both sales and profit
* Product1 is the top-performing product in both sales and profitability
* Justin is the lowest performer across both metrics

## Tools Used

* Microsoft Excel
* SUMIFS, AVERAGE
* Data Cleaning
* Basic Dashboard Design

## Files

* `Sales_Performance_Analysis_Dashboard.xlsx`
* `dashboard.png`

## Notes

This is a beginner-level project focused on learning data analysis fundamentals using Excel.
